# TimerApp 使用说明

## 🚀 快速入门

### 第一步：启动应用

**推荐方式 - 双击启动**
1. 在Finder中找到 `TimerApp.app` 文件
2. **直接双击**应用图标
3. **新功能**：启动时会显示2秒控制界面，便于确认应用已启动
4. 2秒后界面自动关闭，应用隐藏在菜单栏显示时钟图标⏰

**备选方式**
```bash
# 终端启动
cd /Users/wqz/Documents/work/tools/timer
open TimerApp.app

# 或右键点击 -> 打开
```

**启动过程说明**
- ✅ **启动时**：自动弹出控制界面，显示2秒钟
- ✅ **启动后**：界面自动关闭，应用隐藏在菜单栏
- ✅ **验证成功**：看到弹出界面即表示应用启动成功
- ✅ **日常使用**：点击菜单栏 ⏰ 图标打开控制界面

### 第二步：基本使用

1. **点击菜单栏图标** ⏰ 打开控制界面
2. **选择定时时长**：点击预设按钮或输入自定义时间
3. **开始计时**：点击绿色"开始"按钮
4. **查看状态**：菜单栏实时显示剩余时间

## 🎯 详细功能介绍

### 1. 定时时长设置

**快速选择预设时长**
- 15分钟 - 适合短时间休息
- 30分钟 - 番茄工作法标准时长
- 45分钟 - 适合课程或会议
- 1小时 - 长时间任务计时

**自定义时长输入**
1. 点击"自定义"按钮
2. 输入分钟数（如：25）
3. 按回车键确认

### 2. 定时器控制

**启动计时**
- 点击绿色"开始"按钮
- 菜单栏开始显示剩余时间
- 控制界面显示进度条

**暂停计时**
- 点击橙色"暂停"按钮
- 计时暂停，按钮变为"继续"
- 菜单栏显示暂停状态

**继续计时**
- 点击绿色"继续"按钮
- 从暂停位置继续计时

**重置定时器**
- 点击红色"重置"按钮
- 时间重置为初始设置
- 可以重新开始计时

### 3. 自定义音乐提醒

**选择音效文件**
1. 点击"选择音效"按钮
2. 在弹出的文件选择器中找到音频文件
3. 支持格式：MP3、WAV、AIFF、M4A

**重置音效设置**
- 点击"重置"按钮恢复默认音效
- 使用系统默认提示音

**音效状态显示**
- 绿色文字显示已选择的音频文件名
- 灰色文字显示"使用默认音效"

### 4. 菜单栏功能

**实时时间显示**
- 定时器运行时：显示剩余时间（如"15:30"）
- 定时器停止时：显示时钟图标⏰

**快速访问**
- 点击菜单栏图标弹出控制界面
- 支持多任务操作，不影响其他应用

## 🎮 快捷键操作

虽然应用主要使用鼠标操作，但支持以下键盘快捷键：

**控制界面打开时**
- `回车键`：开始/继续计时
- `空格键`：暂停计时
- `R键`：重置定时器
- `ESC键`：关闭控制界面

**自定义时长输入**
- 输入数字后按回车确认
- 支持键盘数字输入

## ❓ 常见问题解答

**Q: 应用启动后在哪里？**
A: 应用在菜单栏显示时钟图标⏰，不会出现在Dock中。

**Q: 如何关闭应用？**
A: 右键点击菜单栏图标选择"退出"，或使用Cmd+Q快捷键。

**Q: 定时结束没有声音？**
A: 检查系统音量设置，或尝试选择其他音效文件。

**Q: 菜单栏时间显示不更新？**
A: 重启应用或检查系统权限设置。
